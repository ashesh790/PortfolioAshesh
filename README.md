# Portfolio Website - Ashesh Parmar

A professional portfolio website showcasing the work and skills of Ashesh Parmar, a Full Stack Software Engineer.

## Technologies Used

- **HTML5** - Structure and content
- **CSS3** - Styling and animations
- **Bootstrap 5.3.0** - Responsive framework
- **JavaScript** - Interactive functionality
- **Font Awesome 6.4.0** - Icons
- **Google Fonts (Poppins)** - Typography

## Features

- ✅ Responsive design that works on all devices
- ✅ Smooth scrolling navigation
- ✅ Animated sections on scroll
- ✅ Interactive skill progress bars
- ✅ Project showcase with hover effects
- ✅ Professional timeline for experience
- ✅ Contact form with validation
- ✅ Modern gradient design
- ✅ Social media links integration

## Sections

1. **Hero Section** - Introduction with call-to-action buttons
2. **About** - Personal information and background
3. **Skills** - Technical skills with progress indicators
4. **Projects** - Featured projects with descriptions
5. **Experience** - Professional work history timeline
6. **Contact** - Contact form and information

## How to Use

1. Open `index.html` in a web browser
2. No build process required - works directly in the browser
3. All dependencies are loaded via CDN

## Customization

### Update Personal Information
- Edit the name, title, and description in `index.html`
- Update contact information in the Contact section
- Modify social media links in the Hero and Footer sections

### Change Colors
- Modify the gradient colors in `styles.css`:
  - Search for `#667eea` and `#764ba2` to change the primary gradient
  - Update other color values as needed

### Add Projects
- Duplicate a project card in the Projects section
- Update the content, technologies, and links

### Update Experience
- Add or modify timeline items in the Experience section
- Update company names, dates, and descriptions
